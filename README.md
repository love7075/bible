# 多版本圣经阅读网站

 目前主流的中文圣经翻译有许多错误，有的是添加译者的个人理解，有的是添加当时流行的神学观点，导致读者无法准确理解圣经的原来的字面意思。
            作者并不懂希伯来语和古希腊语，不过随着当代大语言模型的出现，使得个人追寻原文意思成为可能。
            我参考多个顶级大语言模型（gemini2.5pro chatgpt4,grok3等），对圣经的关键地方，逐字逐词，查找希伯来和古希腊语，利用大模型多个角度探寻原词的意思，并参考英语版翻译，对原文尽最大可能的直译，不做文化转译，不利用中华文化原来有概念进行新的解释。
            最明显是“道”这个字，原文是“logos”，有话语、言语、理性、道理、逻辑等多重含义，强调的是人主观的精神层面
            “道"这个字，在中文语境中，掺杂了儒释道诸子百家的意思，更偏重**自然世界、宇宙、社会**的规律。
            中文圣经翻译者利用“道”这个字来解释“logos”，存在明显的偏差。
            logos英文版一般翻译成”word",相对来说是比较接近原文意思的。
            所以我在翻译的时候，尽量避免使用有文化含义的字， 而是使用更接近原文的字。

## 文件结构
- `index.html`：主页面
- `styles.css`：样式文件
- `script.js`：JavaScript文件

## 使用方法
1. 打开 `index.html` 文件。
2. 从下拉菜单中选择您想要阅读的圣经版本。
3. 页面将显示所选版本的内容。
